# The-Box

## ТЗ: разработать два микросервиса на базе NestJs,  Product-service(карта товара) и Stock-service(склад)  с исползованием такоих инструментов как: Redis, RabbitMQ, Saga, PostgresQL, GraphQL, WebSockets, gRPC.

### Возможное доплнительно использование таких инстурментов как: Loki, Prometheus, Grafana, ClickHouse. 

Создать два микросервиса: **Product-Service (витрина) и Stock-Service (склад)**.<br/> 
Нужно реализовать создание товара и резервирование его на складе через Saga Pattern.
